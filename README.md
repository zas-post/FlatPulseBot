# 🏢 FlatPulseBot

Асинхронный Telegram-бот на **aiogram 3.x** для автоматического семейного поиска недвижимости в Санкт-Петербурге с интеграцией **undetected-chromedriver** для обхода защитных механизмов Авито.

## ✨ Основные фичи
- **Семейный доступ:** Бот отправляет уведомления параллельно всем членам семьи, указанным в конфигурации.
- **Интерактивный UI:** Удобные Inline-кнопки прямо под карточкой квартиры (`🔗 На Авито`, `❌ Удалить`, `⭐️ В семейный чат`).
- **Синхронизация обсуждений:** Возможность в один клик переслать понравившийся вариант в общую семейную группу Telegram.
- **Надежность и обход блокировок:** Использование стратегии загрузки страниц `eager`, динамическое изменение разрешения экрана браузера и автоматические повторные попытки при таймаутах.
- **Архитектура Production-ready:** Разделение логики по модулям (scrapers, bot, database, config), изоляция сессий БД и логирование критических ошибок в файл `errors.log`.

## 📁 Структура проекта

```text
flat_pulse_bot/
│
├── config/               # Конфигурация приложения
│   └── settings.py       # Чтение и валидация переменных окружения
│
├── database/             # Слой работы с базой данных SQLite
│   ├── connection.py     # Безопасный контекстный менеджер сессий
│   └── models.py         # SQLAlchemy модели таблиц
│
├── scrapers/             # Модули парсеров
│   ├── base.py           # Базовый класс с маскировкой ChromeOptions
│   └── avito.py          # Логика скрапинга Авито
│
├── bot/                  # Логика Telegram-бота
│   ├── handlers/         # Обработчики команд и инлайн-кнопок
│   ├── middlewares/      # Прослойка безопасности (AuthMiddleware)
│   └── instance.py       # Инициализация объектов Bot и Dispatcher
│
├── .env                  # Локальные секреты и токены (ИГНОРИРУЕТСЯ GIT)
├── .gitignore            # Правила исключения файлов из репозитория
├── requirements.txt      # Зависимости проекта
└── run.py                # Единая точка входа для запуска приложения
```

⚙️ Локальное развертывание
Клонируйте репозиторий:

``` Bash
git clone [https://github.com/ТВОЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ/FlatPulseBot.git](https://github.com/ТВОЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ/FlatPulseBot.git)
cd FlatPulseBot
```

Настройте виртуальное окружение:

```Bash
python3 -m venv .venv
source .venv/bin/activate  # Для Linux/macOS
pip install -r requirements.txt
```

Создайте файл конфигурации .env в корне проекта:

```
Фрагмент кода
TELEGRAM_TOKEN=твой_токен_бота
ADMIN_ID=8123123432,id_жены,id_сына  # ID пользователей через запятую
FAMILY_CHAT_ID=-1001234567890       # ID общей семейной группы
TARGET_AVITO_URL=ссылка_на_фильтр_авито
AVITO_CHECK_INTERVAL=900            # Интервал проверки в секундах
```

Запустите проект:

```Bash
python run.py
```

🐋 Развертывание в Docker (для Proxmox / VPS)
Проект полностью готов к контейнеризации. Сборка окружения включает в себя автоматическую установку стабильной версии Google Chrome внутри Docker-контейнера.

```Bash
# Запуск сборки и деплоя в фоновом режиме
docker compose up -d --build

# Просмотр логов работы воркера и бота
docker compose logs -f
```
